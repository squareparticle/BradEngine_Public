Variables:

Variables are a way for your program to remember. You can think of them as place holders.

How to create a variable

To store a value into a variable, you first must create a name for your variable. 


    var myVariableName;  // undefined by default

Since it's after 2020, we now use let instead of var. (var is function scoped and let is block scoped) 


    let myVariableName; // undefined by default

Pass the data that you want to start in the variable using the =. 


    let myVariableName = 12;

Note:

Unlike most programming languages, Javascript variables are interchangeable. This means any variable can be a number or a string or true/false value etc... 


    let myVariableName = 12; // now equals 12
        myVariableName = "Brad"; // now equals Brad
        myVariableName = true; // now equals true

Numbers

Javascript has two kinds of numbers (float, int). 


  var a = 42; // integer, int
  var b = -42; // integer, int
  var c = 0.5; // float
  var d = -0.5; // float

Boolean

Booleans are either true or false. 


    var a = true;
    var b = false;

String

A string is a group of characters (words). But they must be inside quotations. 


    var name1 = 'Brad';
    var name2 = "Brad";

Adding variables together

Numbers can use math operators (+ - * /) to add them together 


    let a = 5;
    let b = 2;
    let c = a + b; // 7
    let d = a - b; // -3
    let e = a * b; // 10
    let f = a / b; // 2.5

Strings can be added together (Concatenation) 


    let firstName = 'Brad';
    let lastName = 'Whitten';
    let fullName = firstName + ' ' + lastName;

Short hand operators (++ -- += -= *= /=) 


  let a = 1;
  a++; // a = 2;

  let b = 2;
  b --; // b = 1

  let c = 1;
  c+=5; // c = 6;

  let d = 5;
  d -= 4; // d = 1

  let e = 5;
  e *= 2; // e = 10

  let f = 10;
  f /= 2; // f = 5

Arrays

Arrays are groups of variables stored as one variable

Note:

Use the [ ] to group variables together and comma separate them.


    let numbers = [1,2,3,4];
    let names = ['Brad', 'Dana', 'Adam', 'Brent'];

    let moreNames = [];
    moreNames[0]='Stacey';
    moreNames[1]='Glen';
    moreNames[2]='Steve';

    let evenMoreNames = [];
    evenMoreNames.push('Mike');
    evenMoreNames.push('Lisa');
    evenMoreNames.push('Mark');

Access the data stored in an array by index (A number that points to the location of where your item is stored). Remember: Indexes start at 0 


    let names = ['Brad', 'Dana', 'Adam', 'Brent'];
    console.log(names[2]); // prints Adam to the console

Dictionary (Map / Hashtable / Associative Array)

A dictionary is an array that uses strings (keys) as the index instead of a number.

Note:

Use the { } for dictionary arrays. This is because dictionarys are really just normal objects.


    let numbers = {};
    number['One'] = 1;
    number['Two'] = 2;
    number['Three'] = 3;

    console.log(number['Two']); // 2

Enums

Enums are best used to force the correct spelling for keys in a dictionary and later when using a switch statement.


    const NumberTypes = {
        ONE: 'one',
        TWO: 'two',
        THREE: 'three'
    }
    
    let numbers = {};
    number[NumberTypes.ONE] = 1;
    number[NumberTypes.TWO] = 2;
    number[NumberTypes.THREE] = 3;

    // gives compile error if you spell ONE wrong
    console.log(numbers[NumberTypes.ONE]);

Objects

An object is a single variable that can hold many variables. This is the same as and array but objects can also hold functions. (More about that later)

Note:

Use the { } to create a blank objects. Notice: This is the same as a Dictionary Array because Dictionaries are objects 


    let player = {};
    player.positionX = 100;
    player.positionY = 150;
    player.speed = 10;

Objects can hold Objects inside them. 


    let player = {};
    player.position = {};
    player.position.x = 100;
    player.position.y = 150;
    player.speed = 10;

Remember: This is the same as... 


    let player = {};
    player['position'] = {};
    player['position']['x'] = 100;
    player['position']['y'] = 150;
    player['speed'] = 10;

JSON

JSON is native to JavaScript and therefore lets you save any oject into a string as well as turn any string into an object. This rocks for 2 reasons.

  • Can save the state of your game or program to a database
  • Can create and pass objects on the fly

Note:

Use the { } to create a JSON objects.
Use the : instead of =.
Still use [ ] for arrays
Don't use ; inside of the JSON. Make sure you use commas though 


    let player = {};
    player.position = {};
    player.position.x = 100;
    player.position.y = 150;
    player.speed = 10;
    player.weapons['Gun','Knife'];

    // written in JSON would be

    let player = {
        position{
            x : 100,
            y : 150
        },
        speed: 10,
        weapons:['Gun','Knife']
    };

    // or more compact as
    let player = {position{x:100,y:150},speed:10,weapons:['Gun','Knife']};

String to Object.

Notice: the contents are inside " ". Therefore, javascript just thinks its a bunch of words in a string and not a real object. 


    let string = "{position{x:100,y:150},speed:10,weapons:['Gun','Knife']}";
    let player = JSON.parse(string);

Object to String.


    let player = {};
    player.position = {};
    player.position.x = 100;
    player.position.y = 150;
    player.speed = 10;
    player.weapons['Gun','Knife'];
            
    let string = JSON.stringify(player);

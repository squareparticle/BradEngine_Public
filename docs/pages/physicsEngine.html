PhysicsEngine.js:

The Physics Engine will calculate the location and rotation of a given image, sprite or primitive. This engine was built on top of the Box2D API.

Current repo note:

The active physics demos are in examples/physics. They load shared garage assets from src/sharedMedia/images/garage with findResources(). The current garage files are background.jpg, crate.png, tire.png, run.png, and run2.png.

Starting the Physics Engine

Use the global BradEngine instance to initialize the physics engine. 


    setup(){
        this.physicsEngine=bradEngine.initPhysics()
    }

Loading Resources into the Physics Engine

Load the resources normally and pass them into the physics engine if you want the physics engine to draw the scene for you 


  class MainLevel extends LevelInterface {
    loadResources(){
        this.findResources('physics',['src/sharedMedia/images/garage'])
    }
    setup(){
        this.physicsEngine=bradEngine.initPhysics()
        this.physicsEngine.addImages(this.images)
    } 
  }

Bodies

In physics bodies are the objects that move and collide with each other.

To Add a Body

addBody({name,image,isVisible},shape,isStatic,x,y,{width,height})

name used to know things like which body collided with something or which body did you click on etc...

isStatic if true then this body can not move

shape box, circle - the shape that will be used to calculate the physics with


    setup(){
        this.physicsEngine=bradEngine.initPhysics()
        this.physicsEngine.addImages(this.images)
        this.physicsEngine.addBody({name:'crate',image:'crate.png'},'box',false,500,400,{width:100,height:50})
        this.physicsEngine.addBody({name:'tire',image:'tire.png'},'circle',false,400,300,{radius:25})
        this.physicsEngine.addBody({name:'ground',isVisible:false},'box',true,500,800,{width:1000,height:50})
    }

Debug Mode

This mode is useful to see what Box2D is calculating for the physics


    setup(){
        this.physicsEngine=bradEngine.initPhysics()
        this.physicsEngine.addImages(this.images)
        this.physicsEngine.addBody({name:'crate',image:'crate.png'},'box',false,500,400,{width:100,height:50})
        this.physicsEngine.addBody({name:'tire',image:'tire.png'},'circle',false,400,300,{radius:25})
        this.physicsEngine.addBody({name:'ground',isVisible:false},'box',true,500,800,{width:1000,height:50})
        this.physicsEngine.setupDebug()
    }
    update(delta){
        this.physicsEngine.drawDebug()
    }

Draw Mode

Use the Physics Engine to draw the scene for you.


      setup(){
          this.physicsEngine=bradEngine.initPhysics()
          this.physicsEngine.addImages(this.images)
          this.physicsEngine.addBody({name:'crate',image:'crate.png'},'box',false,500,400,{width:100,height:50})
          this.physicsEngine.addBody({name:'tire',image:'tire.png'},'circle',false,400,300,{radius:25})
          this.physicsEngine.addBody({name:'ground',isVisible:false},'box',true,500,800,{width:1000,height:50})
      }
      update(delta){
        drawImageFrom00('background.jpg',0,0,this.canvas.width,this.canvas.height)
        this.physicsEngine.drawImmediateSimulation()
      }

Note: Drag and Drop

To add Drag and drop to draw mode just add the below line to setup() 


    this.physicsEngine.dragNDrop();

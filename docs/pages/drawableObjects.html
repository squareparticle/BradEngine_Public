Drawable Objects

Drawable Objects are s simple way to organize items in your level.

Life cycle of an Object 


  class myObject{
    constructor{}
    getMouseInput(event){}        
    getMouseMoveInput(event){}
    getKeyboardInput(event){}
    draw(){}
    gameLoop(delta){}
  }

Using a Drawable Object in a Level

Create and Object and add it to the level. 


    class Box{
      constructor(x, y, width, height, color){
        this.x = x;
        this.y = y;
        this.width = width;
        this.height = height;
        this.color = color;
      }
      draw(){
        drawBox(this.x, this.y, this.width, this.height, this.color);
      }
    }
  
    class MainLevel extends LevelInterface {
      setup(){
        this.addObject(`Box`,new Box(50, 50, 150, 50, 'green'));
      }
      draw(){
        this.getObject(`Box`).draw();
      }
    }

Clickable Drawable Object

Use subscribe to attach the object to the level's mouse listener 


    class Box{
      constructor(x, y, width, height, color){
        this.x = x;
        this.y = y;
        this.width = width;
        this.height = height;
        this.userColor = color;
        this.color = color;
      }
  
      getMouseInput(event){
        // change the box to red on mouse down
        if(event.type === "down"){ 
            this.color = 'red';
            this.draw();
        }  
        // change the box to normal color on mouse up
        if(event.type === "up"){
          this.color = this.userColor;
          this.draw();
        }  
      }    
  
      draw(){
        drawBox(this.x, this.y, this.width, this.height, this.color);
      }
    }
    
    class MainLevel extends LevelInterface {
      setup(){
        this.addObject(`Box`,new Box(50, 50, 150, 50, 'green'));
      }
      draw(){
        let button = this.getObject(`Box`).draw();
        this.subscribe(EventTypes.MOUSE,button);
      }
    }

BRADObjects

BRADOBjects are Drawable objects that do more of the work for you. Including handling Sprites.

Life cycle of a BRADObject 


  class myObject extends BRADObject{
    setupParams(){}
    getMouseInput(event){}        
    getMouseMoveInput(event){}
    getKeyboardInput(event){}
    update(delta){}
    draw(){}
  }

Sprites with BRADObjects

Below creates an animating butterfly that animates at 5fps.

Note:

BRADObjects play a sprites idle animation by default 


    class MainLevel extends LevelInterface {
   
      loadResources(){
          this.findResources('physics',['src/sharedMedia/images/anim']);
      }
    
      setup(){
        let butterfly = this.addObject('Butterfly',new Ball(
            {
                canvas:this.canvas,
                width:200, height:200,
                createSprites:[{
                    name:'main',
                    fps:5,
                    image: 'butterfly-a.png',
                    animations: {idle : {startFrame: 0, numFrames:8}},
                    cells: {rows: 1, cols: 8}
                }]
            }
        ));
      }
  
      update(delta){} // required if you want 60fps
    }

Pipeline3D.js:

The Pipeline3D Engine will software render 3D objects using pure math.

Current repo note:

The 3D examples are under examples/3d. Load meshes with findResources(); .x files are parsed with Pipeline3D.parseDirectXFile(), and .obj files are parsed through K3D and converted with Pipeline3D.K3DtoModel().

Loading 3D resources

Resources that can be loaded by the current engine path are DirectX .x meshes and Wavefront .obj meshes. Some examples also load images and JSON-like data through the normal resource loader. 


    loadResources(){ this.findResources('wiremesh','examples/3d/wiremesh/meshes'); }

    setup(){
        this.loadedMeshes['meshes/tank.x'];
    }

Scene Setup

A scene is an collection of 3D objects, lights and cameras used to render to the screen 


    setup(){
        this.scene = new Scene();
        this.scene.camera = {pos:{x:10,y:0,z:25},norm:{x:0,y:0,z:1}, rot:{x:0,y:0,z:0}}
        this.speed = 40; // camera movement and rotation speed
        this.scene.ambientlight = {r:50,g:50,b:50}; // background light when there is no light
    }

Lights

Add lights to the scene 


    let lightPos = {x:-97,y:30,z:-50};
    this.scene.lights.push({pos:lightPos,norm:Tools3D.normalize(lightPos),col:{r:255,g:255,b:255}});

Meshes

Adding a Mesh (3D object) to the scene. 


    let mesh = new MeshObject(this.loadedMeshes['meshes/spaceship.x']);
    mesh.color = {r:255, g:0, b:0};
    mesh.pos={ x:100, y:100, z:500 }
    this.scene.meshObjects.push(mesh);

Render the Scene

Rasterize the polygons into pixels 


    update(delta){  
        this.scene.render(this.canvas);
    }

3D Camera

The defaultCameraKeyboardControls creates a free space camera

Keyboard commands:

"w" MoveCamera.FORWARD
"s" MoveCamera.BACK
"a" MoveCamera.YAXIS_LEFT
"d" MoveCamera.YAXIS_RIGHT
"ArrowDown" MoveCamera.XAXIS_DOWN
"ArrowUp" MoveCamera.XAXIS_UP

runCameraControls(delta, moveSpeed, rotateSpeed) 


    setup(){
        this.scene.camera = {pos:{x:10,y:0,z:25},norm:{x:0,y:0,z:1}, rot:{x:0,y:0,z:0}}
    }

    getKeyboardInput(event){ this.scene.defaultCameraKeyboardControls(event); }

    update(delta){  
        this.scene.runCameraControls(delta, this.speed, this.speed);
        this.scene.render(this.canvas);
    }

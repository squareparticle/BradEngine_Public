Conditions:

Conditions allow the programmer to check if something is true or false

How to create a Condition

Check is something is true

Note:

All code lives between the { } or it's not part of the if statement 


  let a = 2;
  if(a === 2){
    console.log('a = 2'); // displays a = 2
  }

Notice there are 3 === to check if its true.


if(a = 2) <- this would make a = 2
if(a == 2) <- works but should fail if a is not an integer
if(a === 2) <- works and fails if is a is a string (a = '2')

Check is something not (!) true 


  let a = 2;
  if(a !== 2){
    console.log('a = 2'); // will not reach here
  }

if else

When you want to add code to something that is both true and false, you can use an else statement 


  let a = 1;
  if(a === 3){
    console.log('a = 3'); 
  }
  else{
    console.log('a is not 3');  // a is not 3
  }

You can combine multiple if statements together 


  let a = 5;
  if(a === 4 ){
    console.log('a = 4'); 
  }
  else if(a === 3){
    console.log('a = 3'); 
  }
  else if(a === 2){
    console.log('a = 2'); 
  }
  else{
    console.log('a is not 4 or 3 or 2');//a is not 4 or 3 or 2
  }

Combining two conditions

You can check multiple conditions for a single if statement 


  let a = 3;
  if(a > 2 && a < 4 ){
    console.log('a must be 3'); // a must be 3
  }
            

  let a = 4;
  if(a > 2 || a < 4 ){
    console.log('a is either 2 or 4'); // a is either 2 or 4
  }

Truthy / Falsy

Javascript can try and guess what you mean to be true or false.

  • 0 = false vs any number = true
  • undefined variable = false vs a populated variable = true
  • [] = false vs a populated array = true

  let a;
  if(!a){
    console.log('a is not defined'); // a is not definded
  }
            

  let a = 4;
  if(a){
    console.log('a is defined'); // a is defined
  }

Ternary operator

It is possible to shorten an if statement. If it is only one line long then you can remove the { }


  let a = 4;
  if(a===5)
    console.log('a = 5'); 
  else
    console.log('a is not 5'); // a is not 5

You can write an entire if else on one line using ? and : 


  let a = 4;
  // a is not 5
  (a === 5) ? console.log('a = 5') : console.log('a is not 5');

Switch

Instead of a bunch of if else statements, you can use a Switch statement.

Note:

Use switch to select the case statement

Use case as the if statement

Use break to stop the case statement. (Or else it will run the next case statement)

If none of the case statements run then default will be selected 


  let name = 'Brad';
  switch(name){
    case 'Dana':
      console.log('Your name is Dana');
    break;
    case 'Brad':
      console.log('Your name is Brad');
    break;
    default:
      console.log('Your name is '+name);
  }

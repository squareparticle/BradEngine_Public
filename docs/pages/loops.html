Loops:

Loops allow the programmer to loop through a group of code, over and over and over again.

How to create a Loop

Check is something is true

Note:

All code lives between the { } or it's not part of the loop 


  let i = 0;
  // loops 10 times
  while (i < 10) {
    console.log("i = " + i);
    i++;
  }

Use break to exit the loop early. 


  let i = 0;
  while (i < 10) {
    console.log("i = " + i);
    i++;
    if(i === 5) // exits the loop when i = 5
      break;
  }

While loops can be written backwards if needed. This guarantees that the loop runs at least 1 time. 


  let i = 0;
  do{
    console.log("i = " + i);
    i++;
  }
  while (i < 10);

Example of a loop not running the first time 


  // This loop never runs
  let i = 20;
  while (i < 10) {
    console.log("i = " + i);
    i++;
  }

  // ====================

  // This loop will still run the first time
  let i = 20;
  do{
    console.log("i = " + i);
    i++;
  }
  while (i < 10);

For Loops

for loops are the most used loops as they give you lots of control

Note:

You set i to 0 as part of the loop let i = 0;

You check if i is finihed as part of the loop i < 10;

You increase i (by 1 everytime it loops) as part of the loop i++


  for(let i=0; i < 10; i++){
    console.log("i = " + i); // i will be from 0 to 9 
  }

Note:

i only goes up to 9 and not 10.
If you want i to go up to 10 then you need to put an = 


  for(let i=0; i <= 10; i++){
    console.log("i = " + i); // i will be from 0 to 10
  }

Reverse For Loops

Sometimes you want to count down backwards

Note:

You set i to 10 as part of the loop let i = 10;

You check if i is finihed as part of the loop i > 0;

You decrease i (by 1 everytime it loops) as part of the loop i--


  for(let i=10; i >= 0; i--){
    console.log("i = " + i); // i will be from 10 to 0
  }

For Loops (increase by 2's)

Sometimes you want to increase the counter higher then plus one per loop 


  for(let i=0; i <= 10; i += 2){
    console.log("i = " + i); // i will be 0, 2, 4, 6, 8, 10
  }

Advanced For Loop

You can run multiple lines of code as the for loop using commas to separate them.

Explaination of the code:

This code will count up all the numbers in an array and display it to the console.

  • set i to the length of the array AND set r = 0 using ,
  • Use a truthy condition of when (i === 0) to exit the loop
  • Use || to run code that happens when the loop is finished
  • Add all the numbers together in the array with r += arr[i]


  function countArray(arr) {
    for (let i=arr.length, count=0; i-- || console.log(count); count += arr[i]);
  }
  countArray([3,5,0,-2,7,8]); // 21

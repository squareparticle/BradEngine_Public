Classes:

Classes are Objects that you can extend. This is called Inheritance because the child class has all the properties of it's parents class.

For example: Say you wanted an object that can hold any kind of Vehicle. Then you would create a class called Vehicle and it will hold everything that ALL vehicles share, such as, an engine, fuel, topSpeed, etc... Now say you want to create two vehicles with it. (A boat and a truck.) You can sub class Vehicle to create both a Boat and a Car class. (Since trucks are a type of car, it would be better to create a Car class). Cars have 4 wheels and drives on roads. A Truck does the same but it can go off roading. Therefore, you can sub class Car to make a Truck.

Vehicle class

  • has an engine
  • takes fuel
  • has a topSpeed

Car class (extends Vehicle)

  • has an engine
  • takes fuel
  • has a topSpeed
  • has 4 wheels
  • drives on Roads

Truck class (extends Car)

  • has an engine
  • takes fuel
  • has a topSpeed
  • has 4 wheels
  • drives on Roads
  • can go off roading

Boat class (extends Vehicle)

  • has an engine
  • takes fuel
  • has a topSpeed
  • works on water

How to create a Class

Use the Class keyword to create a class.

Note:

Unlike objects, you have to tell javascript that you are making a new class by using the keyword new. This is called instantiation. 


    Class Vehicle{
    }
    let vehicle = new Vehicle();

Passing Variables

You need to create a constructor to pass variables to a class 


  Class Vehicle{
    constructor(engineType){

    }
  }
  let vehicle = new Vehicle('8 cylinder engine');

this (Class variables and functions)

When using variables inside a class you need to use the this keyword. All variables that use this can be used anywhere in the class. 


  Class Vehicle{
    constructor(engineType){
      this.engineType = engineType;
    }
  }
  let vehicle = new Vehicle('8 cylinder engine');

Functions (Methods)

Classes hold functions as well as variables.

Note:

Class functions do not need the keyword function. But to access them you do need to us the keyword this 


  Class Vehicle{
    constructor(engineType){
      this.engineType = engineType;
      this.startEngine(); // start the engine when created
    }
    startEngine(){
      // code to start the engine
    }
    driveForward(){
      // code for driving forward
    }
  }
  let vehicle = new Vehicle('8 cylinder engine');
  vehicle.driveForward();

Sub Class

Create a child class based on the parents properties

Note:

super in the sub class is used to call the code in it's parents class 


  Class Vehicle{
    constructor(engineType, fuel, topSpeed){
      this.engineType = engineType;
      this.fuel = fuel;
      this.topSpeed = topSpeed;
      this.startEngine(); // start the engine when created
    }
    startEngine(){
      // code to start the engine
    }
    driveForward(){
      // code for driving forward
    }
  }

  Class Car extends Vehicle{
    constructor(engineType, fuel, topSpeed){
      super(engineType, fuel, topSpeed); // let Vehicle save these variables
      this.numWheels = 4;
      this.usesRoads = true;
    }
  }

  Class Boat extends Vehicle{
    
    // no need for a constructor as Vehicle already has ome

    driveForward(){
      super.driveForward(); // call the Vehicles drive forward function
      // code for driving forward on water
    }
  }

  let car = new Car('8 cylinder engine', 'gasoline', '200 km/h');
  car.driveForward();

  let boat = new Boat('4 cylinder engine', 'diesel', '50 km/h');
  boat.driveForward();

BRADEngine.js:

(Basic Reusable Application Designer)Engine.js. The BRADEngine works with Levels and BRADObjects

Current repo note:

The engine now lives under src/js, shared art/audio/font assets live under src/sharedMedia, and examples live under examples/. Run the repo with node serveLocal.js or serveLocal.bat, then open http://localhost:8000/.

Level Life Cycle

Levels come with important functions that do most of the work for you.

Note:

You must have a MainLevel 


  class MainLevel extends LevelInterface {
    getKeyboardInput(event){} // talk to keyboard
    getMouseInput(event){} // talk to mouse clicks
    getMouseMoveInput(event){} // talk to mouse move

    loadResources(){} // load images and sounds
    updateLoading(delta, leftToLoad, total){} // loading screen
    setup(){} // setup the level after loading resources
    update(delta){} // update at the configured FPS; delta is seconds since the last update
    draw(){} // draw to the screen at 0 fps
  }

Loading Resources

Load your images and sounds inside of loadResources. When it is done setup will be called.

Note:

use this.findResources('home folder',['resource folders']); to add all the images/sounds you need for this level. Shared media can be loaded with paths such as src/sharedMedia/images/garage. 


  class MainLevel extends LevelInterface {
    loadResources(){
      this.findResources('physics',['src/sharedMedia/images/road','src/sharedMedia/images/garage']);
    }
    setup(){
      // setup your level after all resources have loaded
      this.images['name.png'] // access the loaded images
    } 
  }

While the resources are loading, a loading screen will be presented for you. But you can create your own loading screen using 


  updateLoading(delta, leftToLoad, total){
    // write the code for the loading screen here
  }

Drawing to the Level

Here are the primitive functions for drawing on the level 



  // draw primitives
  const drawBox=(x,y,w,h,col,params)
  const drawRoundBox = (x, y, width, height, col, params)
  const drawCircle=(x,y,r,col,params)
  const drawLine=(x1,y1,x2,y2,col,params)                            
  const drawFill=(path,col, params)

  // draw rotated primitives
  const drawRotatedBox=(x,y,width,height,degrees,col,params)
  const drawRotatedOffsetBox=(x,y,width,height,degrees,col, pivit,params)

  // draw text
  const drawText=(x,y,text,size,col,params)
              
  // draw image - the "image name" needs to be a string key in the level.images array
  // x,y are center of the image and w, h are the radius
  const drawImage=(imgString, x1, y1, w1, h1, degrees)
  const drawImageFrom00=(imgString, x1, y1, w1, h1) // x,y are the top left corner and w, h are the real size
  const drawSprite=(img, x1, y1, w1, h1, x2, y2, w2, h2)
  const drawImageSprite=(imgString, x1, y1, w1, h1, x2, y2, w2, h2, degrees)

  // draw rotated images
  const drawRotatedImage=(image,x,y,width,height,degrees,params)
  const drawRotatedSprite=(image,x1, y1, w1, h1, x,y,width,height,degrees,params)             
  const drawRotatedFlippableImage=(image,x,y,width,height,degrees,params)

  // 3D drawing 
  const drawCircle3D=(x, y, z, r, col, params)
  const drawImage3D=(imgString, x1, y1, z1, w1, h1, degrees)
  const drawSprite3D=(imgString, x1, y1, w1, h1, x2, y2, z1, w2, h2, degrees)
  const drawLine3D=(x1, y1, z1, x2, y2, z2,col, params)

  // builds a new image
  const createFlippedImage=(image, width, height)

Note:

params are used to pass in canvas related parameters when drawing. Example: you can add shadows to text. 


  drawText(0,0,'Some words here',30,'black',{shadowOffsetX:3,shadowOffsetY:3,shadowColor:"rgba(0,0,0,0.3),context.shadowBlur:4"});

draw()
method allows you to draw to the screen 


  draw(){
    drawBox(100,200,50,50,'red');
  }

update()
method allows you to draw to the screen at 60 frames per second. 


  update(delta){
    let x = this.position.x + (speed.x * delta);
    let y = this.position.y + (speed.y * delta);
    drawImage('car1.png', x, y, 128, 128);
  }

Note:

delta is the time between frames. This is needed to keep animations running the same speed even when the frame rate is not stable

Keyboard and Mouse Events

Find out when the player has typed on the keyboard or used the mouse

Note:

You must have a MainLevel 


  class MainLevel extends LevelInterface {
    getKeyboardInput(event){
      console.log(event);
      // type: "down" or "up"
      // key: "h"
      // combined: = Alt and/or Ctrl and/or Shift 
    } 
    getMouseInput(event){
      console.log(event);
      //  type: "down" or "up"
      //  position:
      //    x: 640
      //    y: 572.5
    } 
    getMouseMoveInput(event){
      console.log(event);
      //  type: "moving"
      //  position:
      //    x: 640
      //    y: 572.5
    } 
  }

Network.js:

The Network Engine will send all the the locations and rotations of all objects on the screen to all the clients. This engine was built on top of the PeerJS API.

Current repo note:

The network demos are under examples/network. Network examples need PeerJS and axios loaded by the page, then call bradEngine.initNetwork(callback, fps) and network.startConnection(isServer, stunServer). If META.NETWORK exists, startConnection() will use those values by default.

Starting the Network Engine

host.html
needs to be setup as the server in the META data. By default the connection uses PeerJS defaults unless you pass a custom PeerJS server in STUN_SERVER. 


  const META={ NETWORK:{ IS_SERVER:true, STUN_SERVER:{debug:2}},
               SHOW_DEBUG:true, NETWORK_FPS:60, FPS:60, PHYSICS_FPS:60,
               CREATE_DIR:false }

IS_SERVER: (true/false) - server/client

STUN_SERVER: server used to make the connection

NETWORK_FPS: the number of times per second to send the network data to the clients

Hosting a private STUN server 


  const META={ NETWORK:{ IS_SERVER:true, STUN_SERVER:{host: '54.235.28.16',port: 9000,path: '/myapp'}}}

Use the global BradEngine instance to initialize the network engine. 


  this.network = bradEngine.initNetwork((network input callback function),(network frames per second));
  this.network.startConnection(isServer, stunServer); // arguments are optional when META.NETWORK is defined

Example: 


  setup(){
    this.network = bradEngine.initNetwork(this.getNetworkInput,60);
    this.network.startConnection();
  }

Receiving Data

Clients need to update the screen state before showing all the objects to he screen. The server needs to process the clients user input and then send the screen state back to the clients. 


  getNetworkInput=(data)=>{
    // the client is moving the position of a screen object
    if(this.network.isServer){
        let input = JSON.parse(data);
        this.screenState[input.id].x = input.x;
        this.screenState[input.id].y = input.y;
    }
    // the server is updating all the positions/rotations of
    // all screen objects for this client
    else
        this.screenState = JSON.parse(data);
  }

Sending Data

The server will send all the positions and rotations of all the objects on the screen at 60 fps to all the clients. The clients will only send the user input (mouse/keyboard) to the server for processing. 


  networkUpdate(delta){
    if(this.network.isServer)
        // send the screen state to all the clients
        this.network.sendToAllClients(JSON.stringify(this.screenState));
    else {
        // send the mouse/keyboard input to the server if their was any
        if(this.exportInput) 
            this.network.sendToServer(JSON.stringify(this.exportInput));
    }
  }

Drag and Drop Example

If running on the server then move the object with the mouse. However, if running on the client then remember where to move the object and send that to the server using networkUpdate(delta). 


  getMouseMoveInput(event){
    if(this.dragCar){
        if(this.network.isServer){
            this.dragObject.x = event.position.x;
            this.dragObject.y = event.position.y;
        }
        else{
            this.exportInput={id:this.dragObject.id, x:event.position.x, y:event.position.y}
        }
    }
  }
